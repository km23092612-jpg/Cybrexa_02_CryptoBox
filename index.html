Password Security Tips

🎲

Use a passphrase

4 random words like correct-horse-battery-staple are stronger and easier to remember than P@ssw0rd.

🔑

Use a password manager

Tools like Bitwarden (free) or 1Password let you use a unique strong password for every site without memorizing them.

📱

Enable 2FA everywhere

Two-factor authentication stops attackers even if they have your password. Use an app like Google Authenticator.

🚫

Never reuse passwords

If one site gets breached, attackers try your password everywhere else. Each account needs its own unique password.

📏

Length beats complexity

A 16-character password is exponentially harder to crack than an 8-character one, even with symbols.

🔄

Change after breaches

Check haveibeenpwned.com regularly and change passwords for any breached accounts immediately.
